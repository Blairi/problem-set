# problem-set

Solving some Codeforces problems.

## Compiler used
- g++ 11.3.0

## How to compile
Command used in Linux:
```
$ g++ {problem.cpp} -o {problem}
```
## How to run
Output in console
```
./{problem} < {problem}.in
```
Output in file.out
```
./{problem} < {problem}.in > {problem}.out
```
